# Student-Performance-Predictor
The Student Performance Predictor is a machine learning project developed using Python and Scikit-learn. The system predicts a student’s final marks based on factors such as:  * Study hours * Attendance percentage * Previous academic marks

# Student Performance Predictor

## Project Overview

The Student Performance Predictor is a machine learning project that predicts student final marks using Linear Regression.

The model uses:
- Study hours
- Attendance
- Previous marks

to predict future academic performance.


# Features

- Machine learning prediction
- Student marks forecasting
- Linear regression model
- Simple dataset support


# Technologies Used

- Python
- Pandas
- Scikit-learn


# Installation

Install required libraries:

pip install pandas scikit-learn


# Run Project

python student_performance_predictor.py

# Dataset Columns

- study_hours
- attendance
- previous_marks
- final_marks
